# 📊 Analytics Dashboard Suite

This repository showcases interactive dashboards built using Tableau and Excel to analyze business performance, sales efficiency, and cafeteria operations.

---

## 🔹 Tableau Dashboards

### 📈 Profitability Analysis Dashboard
- Analyzed category-wise profit distribution
- Identified top 10 least profitable products
- Evaluated regional (state-wise) performance trends

### 📞 Sales Performance Dashboard
- Compared incoming and outgoing calls of sales representatives
- Identified top-performing reps based on call activity
- Analyzed peak performance time (2 PM insights)

---

## 🔹 Excel Dashboard

### 🍽 Cafeteria Rating Analysis (CMC Project)
- Developed as Deputy Convenor of Cafeteria Management Committee
- Tracked weekly, monthly, and yearly performance
- Evaluated food vendors based on ratings and feedback
- Supported decision-making to improve food quality and service

---

## 🛠 Tools Used
- Tableau
- Microsoft Excel
- Data Visualization

---

## 📌 Key Insights
- Identified underperforming products affecting profitability
- Highlighted top sales representatives and efficiency patterns
- Improved cafeteria operations using structured rating analysis
